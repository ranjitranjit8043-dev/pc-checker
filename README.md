# Laptop Game Checker

A simple web app that helps you estimate whether a laptop can run popular and demanding games based on CPU, GPU, and RAM scores.

## Run locally

Open [index.html](index.html) in your browser.

## Publish on GitHub

1. Create a new repository on GitHub.
2. Push this project to the repository.
3. In GitHub, open Settings > Pages.
4. Set the source to GitHub Actions.

The included workflow will deploy the site automatically after each push.
